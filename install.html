Install JSCause

The condensed version

curl https://example.com/coming_soon
cd jscause
node jscause

The extended version

JSCause assumes you have NodeJS installed on your system. Please download it and install it before proceeding further.

There are two ways to install JSCause:

1. Download and unzip (preferred)

The preferred way to install JSCause is by downloading the latest zip file, and unzip it in the target installation directory.

  • Download the JSCause zip package.

  • Unzip it in a directory on your system.

  • Open a terminal, change to the target directory.

  • Run node jscause

  • That’s it! If you see INFO: Server 0 listening on port 3000 on the terminal, things went well. If not, please check the troubleshooting section of the documentation.

2. Use npm

  • Create a directory where you want to install JSCause, and switch to it. For example, /home/your_username/jscause: 
    cd ~
mkdir jscause
cd jscause
  • Use npm to install jscause: 
    npm install jscause

  • Run node jscause

  • That’s it! If you see INFO: Server 0 listening on port 3000 on the terminal, things went well. If not, please check the troubleshooting section of the documentation.

Check out the tutorial or the full documentation in order to customize your installation.
